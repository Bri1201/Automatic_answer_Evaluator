About the Model

The NLP Evaluator is an automatic code evaluator based on natural language processing (NLP) models. It assesses the quality of student answers by comparing them to a model answer provided by an educator. The evaluation is done based on the content, semantic similarity, and the presence of specified keywords.

The model uses state-of-the-art NLP techniques such as RoBERTa embeddings and transformer-based models for semantic analysis. It incorporates the following features:

  • Preprocessing: Tokenization, stopword removal, lemmatization
  • Keyword matching and synonym detection
  • Text comparison: Semantic similarity and cosine similarity
  • OCR (Optical Character Recognition) for image inputs

The NLP Evaluator is designed to be user-friendly and can be easily integrated into web applications. Its applications include grading student responses, assessing the quality of written content, and evaluating answers in chatbots and customer service platforms.
